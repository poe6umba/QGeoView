Тестовое задание

Для выполнения тестового задания необходим установленный фреймворк Qt5 и склонированный проект https://github.com/AmonRaNet/QgeoView.

Тестовое задание заключается в расширении библиотеки QGeoView возможностью кэширования онлайн карт.

Необходимо реализовать функционал независимого кэширования загруженных тайлов для каждого провайдера карт, а также предусмотреть возможность принудительной загрузки области карты.

Сохранение тайлов можно реализовать в один отдельный файл или несколько. Для хранения можно использовать базу данных, например, SQLite. Кэш может быть отдельный для каждого провайдера или общий на всех (лучше раздельный).
При сохранении тайла необходимо учитывать название провайдера (поставщик карт, т.е. Google, OSM и т.д), уровень зума и координаты тайла.

Примерная логика работы – при наличии участка карты в кэше происходит его отображение, при его отсутствии происходит загрузка участка и его кэширование, при невозможности загрузки (например, отсутствует Интернет-соединение) отображается пустой тайл с надписью No Data.

Для демонстрации реализованного функционала необходимо разработать приложение-пример с возможностью предварительной загрузки области карты. Должна быть проверена сборка и работоспособность на дистрибутивах GNU\Linux.

Советуем ознакомиться с примерами (samples) и посмотреть класс QGVLayerTilesOnline.
